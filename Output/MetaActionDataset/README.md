# Meta-Action Dataset

This folder saves generated scenario outcomes for SEAN 2.0 meta-action experiments.

## Contents

- scenario_configs.csv: scenario name, robot meta-action, social-force parameter setting, changed parameter values, parameter source, and trajectory file path.
- meta_action_outcome_summary.csv: extracted outcome features for each generated run.
- 	rajectories/: per-timestep trajectory and feature logs generated by MetaActionFeatureLogger.

## Scenario Setup

- Unity scene: Outdoor / CrossPath
- Robot meta-action: fixed meta-action across all runs
- Goal definition: goal is initialized ahead of the robot from its starting pose
- Baseline for pedestrian deviation: default parameter run

## Saved Features

Each trajectory CSV contains per-timestep values including:

- robot position
- pedestrian positions
- distance to goal
- progress along the goal-ahead direction
- current minimum distance to pedestrians
- minimum distance to pedestrians so far
- collision count
- number of pedestrians

The summary CSV contains extracted run-level features:

- minimum distance to pedestrians
- mean pedestrian deviation from default
- maximum pedestrian deviation from default
- progress along the goal-ahead direction
- collision count

## Reproducibility Notes

The default values are confirmed from the current Parameters.cs. The weak/strong robot-repulsion values are recorded from the manual run setup because the temporary parameter changes were not committed to Git. Future runs should save the parameter configuration automatically at runtime.

The default run is used as the baseline for pedestrian deviation, so its pedestrian deviation relative to itself is 0.
